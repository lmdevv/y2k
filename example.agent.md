# example.agent.md

- Use Docker, not host `manimgl`.
- Put new scene files in `backend/videos/agent_sandbox/`.
- Use the sandbox `custom_config.yml`, not the Dropbox-based repo config.
- Render fast first: `-l`, then nicer: `-m` or `-h`.
- For explainer videos, plan first with `manim-composer`.
- For implementation details, load `manimgl-best-practices`.
- Output videos should end up in `videos/`.
